# Data Directory

This directory is used to store local data files required for the project.

## Downloading Data

Populate this folder with the required dataset(s) from the following source:

- 'https://www.eurocontrol.int/dashboard/aviation-data-research'

Example:

```bash
data/
├── README.md
├── dataset.csv
└── other_data_file.parquet
```

## Notes

- The contents of this directory (except this `README.md`) are ignored by Git. Sharing of the data obtained from EUROCONTROL is strictly forbidden.
